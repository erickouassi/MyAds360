Self-hosting: (Plan ahead and schedule your ads)

# How to

Fork this repository https://github.com/erickouassi/MyAds360/fork in your account.

You must host the necessary .js & .json files yourself, and include them in your page directly inside <script> tags.

  • To use by_random.js (Random Ad per site viste) file:
    • Go to the Random Ad by_random.json file.
    • Change to your own advertising campaigns (adLink,adText and cta).
    • View the file by_random.js

{
"id_ad": "7",
"date": "3/3/2022",
"dayText": "Sunday",
"program": "Add program name",
"adLink": "Add your affiliate link",
"adText": "Some text, Some text, Some text, Some text. ",
"cta": "Call to action",
"company": "Company name",
"dateNow": "",
"duration": "90"
}

ðŸ‘‡ Include this in your HTML page.


<div><script async src="./by_random.js"></script>
<div class="randomTextAd"></div></div>
  • To use by_day.js (same ad for the day) file:
    • Go to the Daily Ad by_day.json file
    • Change to your own advertising campaigns on adLink,adText and cta with focus on dayText.
    • View file by_day.js

{
"id_ad": "1",
"date": "3/3/2022",
"dayText": "Monday",
"program": "Add program name",
